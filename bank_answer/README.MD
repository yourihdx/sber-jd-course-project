Модуль, реализующий ответ банка по кредиту.
Данные принимаются в виде JSON, содержащего ID заявки и данные о клиенте, банке, кредитном продукте, условиях кредита.
Модуль реализует асинхронную обработку поступающих заявок.
После обработки направляется ответ в виде JSON, содержащего ID заявки, краткого ответа (да/нет) и развернутого (опционально). 
Запросы принимаются по 
- Spring REST на адрес http://localhost:8086/bankanswer/form_in
- KAFKA topics = "test_sender", groupId = "app.1"
Пример запроса:
{"id":"123-456","bank":"Зеленый банк","sender_addr":"http://localhost:8086/bankanswer/json_in","client":{"full_name":"Иванов Иван Иванович","birth_date":"21.12.1999","phone_Number":"+79101112233","email":"eee@gmail.com","passport":"7001123456"},"loan":{"max_sum":1.0E7,"max_period":60,"percent_rate":5.5,"product_type":"ann"},"loan_offer":{"credit_sum":100000.0,"paymant_day":300,"payment":0.0}}
Важные поля:
id - текстовое, содержит уникальный номер запроса
sender_addr - текстовое, содержит обратный адрес для REST или topic для KAFKA
Желательно заполнить (используется в экранной форме):
bank - имя банка, куда направлен запрос
Пример ответа
{"id":"123-456","res":0,"Comment":"Неверный тип документов"}
Основное окно по адресу:
  http://localhost:8086/bankanswer/data
